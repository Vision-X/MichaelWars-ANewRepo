Topic: Objects in JavaScript

Date 11/17/17

Set the Stage

  1. Setting the stage included setting my phone and alerts to off.
  2. Closed all tabs and programs not involved with this topic/material
  3. Break down the task, set a time block and a timer, get my LEARN ON!
  4. a) Basic principles of an object b) Creating them c) Accessing data in Objects d) Object methods (just a few - not ALL of them) Play

Took the time to build some complex objects, where some keys had values of arrays (of Objects). Spent considerable time playing with dot and bracket notation to suss out which was the best approach to accessing regular data vs complex (array/obj) data with the original Object. -Used Object.assign() to smash multiple Objects together to form a massive hilarious mega Object -Used a constructor function to create a template for a Pet object, and then built New Pet() object instances using the constructor

Reflection

I had some experience with objects, but spending more time diving into best practices and use-cases for each way to create an Object. Beyond that, truly accessing the arrays within the objects was a really great exercise in navigating through and retrieving the values I wanted. I feel much more comfortable knowing the path to accessing the values of these arrays within objects. By far the big takeaway from this learning session was getting familiar with Object methods such as Object.keys, Object.values, Object,assign.... and also using FOR...IN loops on Objects. Being able to join to Objects together could prove useful and would have been awkward to scrub around the documentation to follow this topic. Subtle nuace to these methods and what they can do (and also their limitations) has given me a sharper eye for how to proceed forward with these more difficult Object mutations.
